The Position Property

Exploring the differences between position properties: Absolute, Fixed & Relative

Below are some examples of how each property may be used in postioning.

Without specifying position attributes for the boxes their positions are not affected by top, bottom, left or right properties. They are set by default to static.

Changing an elements position property to relative will allow it to change it's position via the top, left, bottom, right properties.

Absolute positioning will take an element out of the normal flow of the page. Here the green square moves up to take oranges place. Note that my parent box here has it's position set to relative. Therefore setting Orange to right: 10px; kept it inside the square.

This annoying teal box that's been following you down the page has it's position fixed! It has also been taken out of the natural flow of the page just like with Absolute.
